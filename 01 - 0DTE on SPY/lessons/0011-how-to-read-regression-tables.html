บท 0011: อ่าน regression tables ของ paper นี้

บทนี้ไม่สอน econometrics เต็มระบบ แต่สอนว่าแต่ละ table ใน paper ตอบคำถามอะไร และห้ามตีความเกินอะไร

กฎอ่าน table: เริ่มจากคำถามของ table → dependent variable → ตัวแปรหลัก → เครื่องหมาย coefficient → ขนาดผล → ข้อจำกัด

1. Table 1: 0DTE trading ลด volatility ไหม

dependent variable คือ intraday volatility. ตัวแปรหลักคือ 0DTE Trading.

coefficient ลบแปลว่า volatility ต่ำกว่าในวันที่มี 0DTE trading หลังควบคุม fixed effects ตาม specification

2. Table 4: 0DTE เพิ่ม net gamma ไหม

dependent variable คือ market maker net gamma. ตัวแปรหลักยังเป็น 0DTE Trading.

coefficient บวกแปลว่าในวันที่มี 0DTE trading market maker net gamma สูงขึ้น ซึ่งเชื่อมกับ mechanism ที่จะ hedge สวนราคา

4. Table 6-8: mechanism ภายในวัน

Table 6 ดูว่า 0DTE net gamma ทำนาย volatility ในช่วงถัดไปไหม. Table 8 ดูว่า E-mini order flow สวนทาง return มากขึ้นเมื่อ net gamma สูงไหม

สอง table นี้สำคัญเพราะไม่ได้แค่บอก effect ระดับวัน แต่สนับสนุนว่า hedging flow ภายในวันเป็นกลไก

5. ดาวไม่ใช่ทุกอย่าง

ดาวบอก statistical significance แต่การตีความต้องดู design, direction, timing, mechanism และ limitation ด้วย

โดยเฉพาะ E-mini evidence ต้องจำว่า circumstantial ไม่ใช่ direct tracking ของ market maker trades
