Trading Plan And Journal page ระบุว่า plan ต้องมี hypothesis, market, instruments, timeframe, data source, entry/exit, sizing, risk limits, expected holding period, backtest summary, weaknesses, execution assumptions, pause/reduce/retire rules.

Governance rules

  • เปลี่ยน rule ได้เฉพาะรอบ review ไม่ใช่ระหว่างตลาดเปิด
  • ทุก change ต้องมี rationale และ before/after test
  • เพิ่ม size ได้เฉพาะเมื่อผ่าน live-forward sample gate
  • ถ้าเกิด execution incident ให้ pause จน root cause ชัด
  • ถ้าผลจริงหลุด confidence band ให้ลด size หรือหยุด

Final retrieval

  1. ระบบพร้อม live ต้องผ่าน gate อะไรบ้าง
  2. journal แยก strategy loss กับ process error อย่างไร
  3. เมื่อไหร่ควร retire strategy
