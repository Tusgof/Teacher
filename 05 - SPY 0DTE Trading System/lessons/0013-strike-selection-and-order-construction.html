Strike selection และ order construction

การเลือก strike และการส่ง order คือจุดที่ edge เล็ก ๆ หายไปได้ทันที

Strike gap lesson

Strike Selection page จาก ORB source แยก breakout-to-long-strike gap เป็นตัวแปรสำคัญ: gap 0.96-2.00 ดีขึ้นใน simulation แต่ยังต้อง validate บน real option-chain data.

Strike selection dimensions

มิติคำถาม
Dollar gapbreakout price ห่าง long strike เท่าไร
Deltaspread ตอบสนองต่อ underlying แค่ไหน
Debitต้องใช้เงินและ target/stop เท่าไร
Width$1, $2, $5 spread width เปลี่ยน payoff อย่างไร
Liquiditybid/ask และ size พอไหม

Order construction

  • ใช้ spread order ถ้า broker รองรับ เพื่อควบคุม net debit
  • limit price ต้องตั้งตาม liquidity rule ไม่ chase market
  • ถ้าไม่ fill ภายในเวลาที่กำหนด ให้ cancel ไม่ใช่ไล่ราคาไม่จำกัด
  • OCO/exit orders ต้องสอดคล้องกับ actual fill debit
  • time stop ต้องเป็น hard rule

Validation

แยกผล backtest ตาม gap bucket, debit bucket, spread width, entry time และ bid/ask width เพื่อรู้ว่า edge มาจากอะไร.

Retrieval

  1. strike gap ส่งผลต่อ debit และ delta อย่างไร
  2. ทำไม spread order สำคัญ
  3. ควร log bucket อะไรเพื่อวิเคราะห์ strike selection
