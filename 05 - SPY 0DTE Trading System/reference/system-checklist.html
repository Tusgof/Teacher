SPY 0DTE System Checklist

reference sheet สำหรับใช้ตรวจระบบก่อน research, paper trading, และ live rollout

1. Hypothesis Gate

  • ระบุ edge hypothesis ชัดเจน
  • แยก evidence type: paper insight, simulation, backtest, paper trade, live-forward
  • เขียนเหตุผลว่าทำไม edge อาจ persist

2. Data Gate

  • มี SPY intraday bars และ option bid/ask chain
  • timestamp/timezone ตรวจได้
  • event calendar และ VIX available ก่อน entry
  • ไม่มี feature ที่รู้หลัง decision time

3. Backtest Gate

  • event-driven backtest
  • bid/ask + slippage + fees
  • trade log และ skipped-trade log
  • parameter sensitivity และ subperiod analysis
  • OOS/walk-forward results แยกจาก in-sample

4. Risk Gate

  • per-trade max loss
  • daily/weekly/monthly loss limit
  • ES/cVaR, worst day, max drawdown
  • Monte Carlo ruin probability
  • kill switch และ pause rule

5. Live Gate

  • paper trading ผ่าน workflow
  • orders/fills/rejections logged
  • TP/SL/time stop ทำงานจริง
  • dashboard/alerts พร้อม
  • เริ่ม live ด้วย reduced size เท่านั้น
