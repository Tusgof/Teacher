Equity premium และ predictability

paper แสดงว่า positive return states กลับดึง equity premium ลงใน 0DTE

Equity premium decomposition

paper แยกว่า equity premium มาจาก return state ช่วงไหน พบว่าส่วนใหญ่ของ 0DTE มาจาก return ติดลบถึงใกล้ศูนย์ แต่ positive returns contribute ติดลบ

Predictability

variance risk premium ทำนาย intraday excess returns ด้วยค่าสัมประสิทธิ์ลบ โดย driven จาก upside component แปลว่าเมื่อ upside risk compensation สูง future market return มักต่ำลง

ถามตัวเอง

  1. positive returns contribute ติดลบหมายความว่าอะไร
  2. ทำไม upside VRP ที่สูงจึงสัมพันธ์กับ expected return ต่ำ
