Payoff, PNL, Return: สามคำที่ต้องแยกก่อน

ถ้าแยกสามคำนี้ไม่ได้ ตารางของ paper จะหลอกตาเราได้ง่ายมาก

จำสั้น ๆ: payoff คือสิ่งที่ option จ่ายตอนหมดอายุ, PNL คือ payoff ลบต้นทุนเข้า position, return คือ PNL เทียบกับเงินที่จ่ายซื้อ option หรือ premium

1. Payoff คือปลายทาง

สำหรับ call payoff จะดีเมื่อ S&P 500 ปิดเหนือ strike ส่วน put payoff จะดีเมื่อ S&P 500 ปิดต่ำกว่า strike

paper เปิด position ตอน 10:00 ET และถือไปถึง 16:00 ET เป็น baseline หลัก

2. PNL คือสิ่งที่เราควรสนใจมากกว่า

ถ้า option จ่าย 10 แต่ตอนเข้า position เราจ่าย 8 PNL คือบวก 2 ถ้าจ่าย 12 PNL คือขาดทุน 2

paper รายงาน PNL เป็นเปอร์เซ็นต์ของ underlying เพื่อให้ option และ strategy หลายแบบเทียบกันได้

3. Return อาจดูเว่อร์ใน 0DTE

option ใกล้หมดอายุบางตัวมีราคาเล็กมาก ถ้า payoff ขยับนิดเดียว return เทียบ premium อาจดูใหญ่มาก แต่ PNL ต่อ underlying อาจเล็ก

ดังนั้น paper ใช้ทั้ง return และ PNL แต่สำหรับ multi-leg strategies มักเน้น PNL scaled by spot มากกว่า

4. Retrieval

Payoff กับ PNL ต่างกันอย่างไร?

payoff คือเงินที่จ่ายตอน expiry ส่วน PNL คือ payoff ลบต้นทุนเข้า position

ทำไม return ของ OTM 0DTE อาจ misleading?

เพราะ premium เล็กมาก ทำให้ return percentage ใหญ่ แม้ PNL ต่อ underlying จะเล็ก
