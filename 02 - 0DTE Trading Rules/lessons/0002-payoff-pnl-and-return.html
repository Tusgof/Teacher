บท 0002: Payoff, PNL, Return แบบจับต้องได้

ก่อนเข้า VRP หรือ strategy tables ต้องเข้าใจว่า paper วัดกำไรขาดทุนอย่างไร

จำก่อน: Payoff คือสิ่งที่ option จ่ายตอนหมดอายุ. PNL คือ payoff ลบราคาที่จ่ายตอนเข้า. Return คือ PNL หารด้วยเงินที่จ่ายซื้อ option. ใน 0DTE, return อาจดูใหญ่เว่อร์เพราะ premium เล็กมาก

1. ตัวอย่าง call ง่าย ๆ

สมมติ 10:00 ET, S&P 500 อยู่ที่ 5,000 จุด คุณซื้อ call strike 5,010 ที่ราคา 4 จุด และถือถึงปิดตลาด

กรณีปิดที่ 5,030

call payoff = 5,030 - 5,010 = 20 จุด. PNL = 20 - 4 = +16 จุด

กรณีปิดที่ 5,005

call หมดค่า. payoff = 0. PNL = 0 - 4 = -4 จุด

2. ทำไม paper ชอบเทียบกับ underlying

ถ้า S&P 500 อยู่ที่ 5,000 จุด PNL +16 จุด เทียบกับ underlying คือ 16 / 5,000 = 0.32%

paper ใช้วิธี scale แบบนี้เพื่อให้ option หลาย strike และ strategies หลายแบบเทียบกันได้ ไม่ใช่ให้ premium เล็ก ๆ ทำให้ return ดูใหญ่เกินจริง

3. Return อาจหลอกตาใน 0DTE

จากตัวอย่างเดิม คุณจ่าย 4 จุดแล้วได้ PNL +16 จุด return ต่อ premium คือ +400% ดูน่าตื่นเต้นมาก

แต่ในมุม portfolio ที่อ้างอิง underlying 5,000 จุด มันคือ +0.32% ของ underlying เท่านั้น ทั้งสองเลขถูกทั้งคู่ แต่ตอบคนละคำถาม

Return ต่อ premium: เหมาะกับดู leverage ของ option แต่ไวต่อ premium เล็กมาก
PNL ต่อ underlying: เหมาะกับเทียบ strategy หลายแบบใน paper นี้

4. multi-leg strategy คือบวก PNL ของแต่ละขา

ถ้า strategy มี 2 ขา เช่น ซื้อ call หนึ่งตัวและขาย call อีกตัว PNL ของ strategy คือ signed sum ของ PNL ทุก leg

คำว่า signed สำคัญ: leg ที่ long กับ leg ที่ short มีเครื่องหมายตรงข้ามกัน

5. เชื่อมกับ paper

paper สร้าง option grid ด้วย moneyness เช่น 0.99, 1.00, 1.01 แล้วคำนวณ payoff ถึง 16:00. จากนั้นคำนวณ PNL = payoff - mid price และรายงานเป็น percentage of underlying

แปลว่าเมื่อคุณเห็น Table ที่บอก mean PNL เป็น bps หรือ % of underlying ให้คิดว่าเป็นกำไรขาดทุนของ strategy เทียบกับระดับดัชนี ไม่ใช่ return ต่อเงิน premium เสมอไป

6. Retrieval

ถ้าซื้อ call ราคา 4 จุด แล้วหมดอายุไร้ค่า PNL คือเท่าไร?

-4 จุด เพราะ payoff เป็นศูนย์และคุณจ่าย premium ไป 4 จุด

ทำไม return ต่อ premium ของ 0DTE อาจดูแรงมาก?

เพราะ premium เหลือน้อยมาก ตัวหารเล็ก ทำให้เปอร์เซ็นต์ return ใหญ่ แม้ PNL ต่อ underlying ไม่ใหญ่มาก

paper เน้น PNL scaled by spot เพราะอะไร?

เพื่อให้ option และ multi-leg strategies ต่าง strike ต่างราคา เทียบกันได้ในหน่วยเดียว
