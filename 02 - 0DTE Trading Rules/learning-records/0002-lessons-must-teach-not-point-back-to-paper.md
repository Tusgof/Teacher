# Lessons must teach, not point back to paper

The user corrected the course design: lessons should make the paper understandable from zero inside the lesson itself, not summarize and then tell the user to read the paper. For 0DTE Trading Rules and future paper courses, each lesson should explain core terms with concrete examples, connect tables to the question they answer, and avoid assuming knowledge of VRP, OOS testing, skewness, ES1%, or option strategy payoff shapes.
